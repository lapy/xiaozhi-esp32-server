# RAGFlow API Reference Index

This folder contains an internal English reference for the RAGFlow-facing
knowledge-module APIs used by the downstream integration. The goal is to keep a
 practical map of the upstream surface area without carrying Chinese-only
 documentation into the downstream branch.

## Scope

- Dataset and document management
- File management
- Chat assistants and chat sessions
- Agent and SearchBot integrations
- Dify-compatible retrieval
- Supplemental session and debugging endpoints

  instead of reintroducing large translated copies.
